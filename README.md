# nextrapi-tinkering

## Arrancar el proyecto

### CMS (Strapi)

En la carpeta `./cms`:

```sh
yarn
yarn develop
```

Acceder desde el navegador a: http://localhost:1337

### Web (Next.js)

En la carpeta `./web`:

```sh
yarn
yarn dev
```

Acceder desde el navegador a: http://localhost:3000
